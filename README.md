# FocusManager PWA v1.3.2 Elapsed Time Indicator

Session 스타일의 개인 집중 관리 PWA입니다.  
핵심 흐름은 **의도 설정 → 집중 세션 실행 → 회고 기록 → 타임라인 정리 → 리포트 분석 → 공부 계획/목표 관리 → 백업**입니다.

손해사정사 공부, 장시간 학습, 업무 집중, 하루 시간 사용 점검을 목적으로 합니다.

## 현재 버전 핵심

v1.3.2는 v1.3.1 Persistence Hotfix를 기준으로, 앱을 켜고 타이머를 사용하는 동안 **상단 고정 시계에서 현재 세션의 경과 시간/남은 시간/종료 예정 시각을 바로 확인할 수 있도록 개선한 버전**입니다.

Apple Calendar 연동은 제외하고, FocusManager 내부에서 날짜별 공부 계획과 목표를 관리합니다.

이번 버전은 기존 저장 안정화 구조를 유지하면서 화면 상단 고정 배지를 확장했습니다. 세션이 없을 때는 현재시간만 작게 표시하고, 세션 중에는 경과/남은 시간/종료 예정 시각/30분 단위 경과 상태를 함께 표시합니다.

- 상단 고정 현재시간 배지 유지
- 세션 중 경과 시간 표시
- 세션 중 남은 시간 표시
- 목표 시간 초과 시 초과 시간 표시
- 종료 예정 시각 표시
- 30분 단위 경과 상태 표시
- 일시정지 상태 표시
- 오늘 총 목표 시간 설정
- 주간 목표 시간 설정
- 과목별 일일/주간 목표 설정
- 날짜별 공부 계획 추가
- 계획 세션 목록 표시
- 계획에서 바로 집중 세션 시작
- 계획을 세션 화면에 불러오기
- 계획 대비 실제 달성률 표시
- 대시보드에 오늘 계획/목표 달성률 표시
- JSON/GitHub 백업에 계획 데이터 포함
- 기존 GitHub 백업, 모달 핫픽스, 저장 안정화 유지
- 저장 키 `focus_manager_v1` 유지
- 저장 안정화: mirror/last-good localStorage 복구 슬롯 유지
- 저장 안정화: IndexedDB 안전 복사본 유지
- 앱 종료/백그라운드 전환 시 추가 저장 유지

## 포함 기능

### 대시보드

- 오늘 집중 시간
- 세션 수
- 최장 세션
- 가장 많은 카테고리
- 시험 D-day 씰
- 현재 세션 요약
- 오늘 계획 요약
- 오늘 목표/계획 대비/이번 주 목표 달성률

### 상단 고정 세션 표시

- 세션이 없을 때: 현재 시간과 날짜만 표시
- 세션 중: 경과 시간, 남은 시간, 종료 예정 시각 표시
- 목표 시간을 넘기면 남은 시간 대신 초과 시간 표시
- 30분/1시간/1시간 30분처럼 30분 단위 경과 상태 표시
- 일시정지 중에는 일시정지 상태 표시
- 스크롤을 내려도 상단에서 계속 확인 가능

### 세션

- 의도/할 일 입력
- 과목/카테고리 선택
- 세션 타입 선택
  - 이론 / 기출 / 암기 / 오답 / 답안작성 / 복습 / 기타
- 회독 단계 선택
  - 미지정 / 1회독 / 2회독 / 3회독 / 4회독 / 5회독+ / 최종정리
- 파트/범위 입력
- 25분/50분/90분/자유 세션
- 시작/일시정지/재개/종료
- 목표 시간 도달 알림
- 종료 회고 메모
- 집중/보통/산만 상태 기록

### 플래너

- 날짜별 계획 추가
- 시작/종료 시간 또는 목표 분 입력
- 과목/카테고리 지정
- 세션 타입/회독/파트 기록
- 계획 상태 표시
- 계획에서 세션 바로 시작
- 계획 삭제

### 리포트

- 7일 / 14일 / 30일 기준 분석
- 카테고리별 집중 시간
- 공부 방식 분포
- 회독 현황
- 요일별 패턴
- 시간대별 집중 패턴

### 설정

- 하루 목표 시간
- 시험일 D-day
- 알림음 설정
- 회고 입력 유도 설정
- 카테고리 추가/삭제
- JSON 백업/복원
- GitHub 백업/복원
- 세션 종료 후 자동 GitHub 백업 옵션
- GitHub 토큰 삭제

## 백업

v1.3.2의 JSON/GitHub 백업에는 다음 데이터가 포함됩니다.

- settings
- categories
- sessions
- activeSession
- plans
- 과목별 목표
- 주간 목표

GitHub token은 백업 파일에 포함하지 않습니다. 각 기기의 로컬 저장소에만 보관합니다.

## 배포

GitHub Pages 기준으로 저장소 루트에 다음 파일이 있어야 합니다.

```text
index.html
styles.css
app.js
manifest.webmanifest
sw.js
icons/
README.md
CHANGELOG.md
PROJECT_STATUS.md
```

업데이트 후 PWA 캐시가 남아 있으면 주소 뒤에 다음 값을 붙여 확인합니다.

```text
?v=1.3.2
```
