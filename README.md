# FocusManager PWA v1.3.1 Persistence Hotfix

Session 스타일의 개인 집중 관리 PWA입니다.  
핵심 흐름은 **의도 설정 → 집중 세션 실행 → 회고 기록 → 타임라인 정리 → 리포트 분석 → 공부 계획/목표 관리 → 백업**입니다.

손해사정사 공부, 장시간 학습, 업무 집중, 하루 시간 사용 점검을 목적으로 합니다.

## 현재 버전 핵심

v1.3.1은 v1.3.0 Study Planner를 기준으로 **앱 재실행 후 데이터가 초기화되는 문제를 방지하는 저장 안정화 핫픽스**를 추가한 버전입니다.

Apple Calendar 연동은 제외하고, FocusManager 내부에서 날짜별 공부 계획과 목표를 관리합니다.

이번 핫픽스는 저장 안정성을 위해 localStorage 기본 저장 외에 mirror/last-good 복구 슬롯과 IndexedDB 안전 복사본을 추가했습니다. 앱을 껐다 켰을 때 기본 저장값이 비어 있거나 손상되면 가능한 경우 안전 복사본에서 자동 복구합니다.

- 오늘 총 목표 시간 설정
- 주간 목표 시간 설정
- 과목별 일일/주간 목표 설정
- 날짜별 공부 계획 추가
- 계획 세션 목록 표시
- 계획에서 바로 집중 세션 시작
- 계획을 세션 화면에 불러오기
- 계획 대비 실제 달성률 표시
- 대시보드에 오늘 계획/목표 달성률 표시
- JSON/GitHub 백업에 계획 데이터 포함
- 기존 GitHub 백업, 고정 시계, 모달 핫픽스 유지
- 저장 키 `focus_manager_v1` 유지
- 저장 안정화: mirror/last-good localStorage 복구 슬롯 추가
- 저장 안정화: IndexedDB 안전 복사본 추가
- 앱 종료/백그라운드 전환 시 추가 저장

## 백업

v1.3.1의 JSON/GitHub 백업에는 다음 데이터가 포함됩니다.

- settings
- categories
- sessions
- activeSession
- plans
- 과목별 목표
- 주간 목표

GitHub token은 백업 파일에 포함하지 않습니다. 각 기기의 로컬 저장소에만 보관합니다.

## 배포

GitHub Pages 기준으로 저장소 루트에 다음 파일이 있어야 합니다.

```text
index.html
styles.css
app.js
manifest.webmanifest
sw.js
icons/
README.md
CHANGELOG.md
PROJECT_STATUS.md
```

업데이트 후 PWA 캐시가 남아 있으면 주소 뒤에 다음 값을 붙여 확인합니다.

```text
?v=1.3.1
```
