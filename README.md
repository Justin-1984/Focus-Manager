# FocusManager PWA v1.0.0 Initial

Session 스타일의 개인 집중 관리 PWA 초기 버전입니다.  
핵심 구조는 **의도 설정 → 세션 실행 → 회고 기록 → 타임라인/리포트 분석**입니다.

## 포함 기능

- 대시보드
  - 오늘 집중 시간
  - 세션 수
  - 최장 세션
  - 목표 달성률
  - 최근 기록
- 세션
  - 의도/할 일 입력
  - 카테고리 선택
  - 25분/50분/90분/자유 세션
  - 시작/일시정지/재개/종료
  - 종료 회고 메모
  - 집중/보통/산만 기분 기록
- 타임라인
  - 날짜별 세션 시간표
  - 진행 중 세션 표시
- 리포트
  - 최근 7/14/30일 집중 시간 그래프
  - 카테고리별 분포
  - 집중 시간대/평균 세션/집중 비율
- 설정
  - 하루 목표 시간
  - 카테고리 추가/삭제
  - JSON 백업/복원
  - 전체 초기화
- PWA
  - manifest 포함
  - service worker 포함
  - iPhone/iPad/PC 반응형 UI
  - 로컬 저장 기반

## 현재 제한

- v1.0은 로컬 저장 기반입니다.
- 여러 기기 실시간 동기화는 아직 없습니다.
- PWA만으로는 Mac 앱 차단, 웹사이트 차단, Slack 자동 음소거 같은 시스템 제어는 어렵습니다.
- Apple Shortcut 연동은 후속 버전에서 별도 URL Scheme/간단 API 구조를 설계해야 합니다.
