# FocusManager PWA v1.3.8 Planner Layout Rebuild

v1.3.8은 v1.3.7을 기준으로 플래너 화면의 모바일·태블릿·PC 카드 배치 구조를 재정렬한 UI 핫픽스입니다.

## 수정 내용

- 플래너 상단 날짜 선택 영역 정렬 통일
- 데스크톱에서 계획 복사/계획 추가는 왼쪽 열, 목표 설정은 오른쪽 열로 고정
- 태블릿·모바일에서는 카드 순서를 한 줄 흐름으로 재배치
- 목표 설정 카드 높이 때문에 다른 카드 아래에 큰 빈 공간이 생기던 문제 수정
- 날짜 입력, 복사 버튼, 시작/종료/목표 분 입력 폭 보정
- 모바일에서 플래너 요약 카드와 입력 필드를 안정적으로 1열 배치
- 기존 저장 키 `focus_manager_v1` 유지
- service worker cache `focusmanager-v1-3-8`
